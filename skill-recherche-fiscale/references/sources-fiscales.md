## 3. BOFiP — Structure de navigation

Le BOFiP est organisé par impôt :
- **IR** : BOFiP > IR > (sous-parties par revenu)
- **IS** : BOFiP > IS > (sous-parties)
- **TVA** : BOFiP > TVA
- **Droits d'enregistrement** : BOFiP > ENR
- **Procédures** : BOFiP > CF (Contrôle fiscal)

> Via Open Legi : rechercher "BOFiP [sujet]" ou l'identifiant BOFiP si connu.
